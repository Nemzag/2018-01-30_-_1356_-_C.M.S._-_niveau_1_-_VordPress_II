Allows administrators to disable comments on their website. Comments can be disabled according to Post, Page, Media type.
Administrators can also DISABLE or ENABLE there individual post and pages if they want.

== Description ==

Disable Comments WPZ is a powerful and easy to use plugin, it allows administrators to globally or individually disable comments on any post type in one click (posts, pages, attachments(media), etc.)

<blockquote>
But sometimes you feel that some specific posts comments should be Turned on, this following feature is available in the plugin, You can Enable/Disable comments for any particular post/page by clicking on check box in 'DISABLE POSTS' Tab or 'DISABLE PAGES' Tab. 
</blockquote>


== Main Features ==

<blockquote>
1. Easy Enable or disable Comments <br>
2. Disable Comments Globally <br>
3. Disable Comments on Certain Pages <br>
4. Disable Comments on Posts Only <br>
5. Disable Comments on Pages Only <br>

</blockquote>

== Installation ==

1.  Upload `Disable Comments | WPZest` folder to the `/wp-content/plugins/` directory
2.  Activate the plugin through the 'Plugins' menu in WordPress
3.  The plugin settings can be accessed via the 'Settings' menu in the administration area.
